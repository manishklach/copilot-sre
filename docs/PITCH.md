# GitHub Positioning

## Repo tagline

Copilot SRE turns GitHub Copilot CLI into an incident commander.

## GitHub description

An incident-response cockpit built on top of GitHub Copilot CLI that correlates telemetry, GitHub activity, runbooks, mitigation plans, rollback guidance, and a Copilot-ready investigation brief.

## Short elevator pitch

Copilot SRE gives Copilot CLI the operational context layer it is missing today. Instead of starting from a vague prompt during an outage, it assembles alerts, logs, metrics, deploy history, repo activity, and runbooks into one incident brief, then hands that to Copilot for faster diagnosis and safer mitigation.

## Why Microsoft should care

- It extends Copilot CLI instead of competing with it.
- It creates a natural GitHub + Azure story.
- It is enterprise-friendly because the workflow is inspectable and operator-driven.
- It demonstrates how Copilot can move from coding assistant to operational teammate.

## Demo note
